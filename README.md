# Terraform WordPress Setup

Este projeto configura uma VM na Azure que instala Docker e sobe um container com WordPress.

## Requisitos

- Terraform instalado
- Conta na Azure
- Credenciais configuradas (usando Azure CLI ou variáveis de ambiente)
